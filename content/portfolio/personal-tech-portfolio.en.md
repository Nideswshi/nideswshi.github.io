## Project Goal

The main goal of this project is to turn a template-like blog into a site that works better for personal branding, technical writing, and career presentation.

It needs to support three scenarios at the same time:

- act as a portfolio for selected projects
- act as a technical blog for notes and reflections
- act as a public-facing resume landing page

## What I Worked On

- information architecture reorganization
- homepage and portfolio redesign
- light and dark theme unification
- GitHub Pages deployment workflow repair
- PWA and base experience improvements

## Design Decisions

This version deliberately avoids noisy visual effects and focuses on:

- restrained color usage
- stronger typography hierarchy
- cleaner spacing and layout rhythm
- a structure that can keep growing with new work and writing
